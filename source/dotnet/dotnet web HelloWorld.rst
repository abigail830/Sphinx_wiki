Dotnet Web HelloWorld
==========================

Create Web App With CLI
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block::

  dotnet new webApp -o myWebApp --no-https
  
Run Web App With CLI
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block::

  dotnet run

Default startup link would be : http://localhost:5000/
